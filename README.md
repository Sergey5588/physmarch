Raymarching game engine made by me and my friend [@MAX-TS](https://github.com/MAX-TS)
## compile from source

`git clone https://github.com/Sergey5588/physmarch`
`cd physmarch`
`mkdir build`
`cd build`
`cmake ..`
`make`
